Three new content blocks:

  • Announcements — admin posts a message (Firestore `announcements` collection). Gold left border, warm yellow bg. Shows author + time. Dismissable or auto-expires.
  • "This Week's Focus" — field on the session doc: `topic` and `description`. Plum gradient card. Links to related tutorial. Admin sets it when creating or editing a session.
  • Quick Actions — 3 shortcut tiles: Pay, Add Photo, Invite. Context-aware (Pay only shows if unpaid, etc.)

Order on home: Session hero → Announcement (if any) → This Week's Focus (if set) → Quick Actions → thick divider → Gallery feed
