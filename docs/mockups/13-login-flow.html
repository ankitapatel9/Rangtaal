Design notes:

  • Login — logo + tagline centered in hero area, phone input at bottom. Single "Continue" CTA. No social login (phone auth only).
  • OTP — individual digit boxes with gold focus ring. Auto-submits on 6th digit. Resend link below. Back arrow returns to login.
  • Welcome — only shows for new users (no Firestore user doc yet). Collects name only. "Join Rangtaal" creates the user doc and redirects to home.
  • All three use the Plum & Gold palette. Warm ivory background, white inputs, gold CTAs.
